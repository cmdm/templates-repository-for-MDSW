# Specifications

Templates for the Specifications phase.

| Template | Description |
|---|---|
| [Software Requirements Specification](software-requirements-specification.md) | Template for documenting software functional and non-functional requirements, interfaces, security, usability, and traceability. |
| [Software Requirements Specification Review Report](software-requirements-specification-review-report.md) | Template for recording the results of the formal SRS review, including review checklist, findings, and approval verdict (IEC 62304 §5.2.6). |
